git init:

git, el cual contiene todos los archivos necesarios del repositorio

git fetch:
Descarga los cambios realizados en el repositorio remoto.

git commit -am "<mensaje>":
Confirma los cambios realizados.
 El “mensaje” generalmente se usa para asociar al commit 
una breve descripción de los cambios realizados.

git push origin <nombre_rama>:
Sube la rama “nombre_rama” al servidor remoto.

git status:
Muestra el estado actual de la rama, como los cambios que hay sin commitear.

git add <nombre_archivo>:
Comienza a trackear el archivo “nombre_archivo”.
